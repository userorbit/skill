Setup

Claude Code

claude install-skill https://github.com/userorbit/skill

Codex

git clone https://github.com/userorbit/skill.git ~/.codex/skills/userorbit

Any other agent

Clone the repo and point your agent's context at userorbit/SKILL.md. There's also a hosted MCP server at https://api.userorbit.com/mcp if your tool speaks MCP.

Authentication

Grab an API key and team ID from Settings → API in your Userorbit workspace, then:

export USERORBIT_API_KEY="uo_..."
export USERORBIT_TEAM_ID="..."

Read-only keys can call list/info/count/search endpoints; read/write keys can call everything.

How it works

Every Userorbit resource is a POST endpoint at https://api.userorbit.com/api/v1/<resource>.<action>. The skill gives your agent a compact map of all of them — 150+ endpoints — plus required fields, enums, and common workflows, so it can chain calls correctly: find the collection, create the announcement, publish it.
