What changes

Product ops you never do by hand again

Your agent already knows what shipped — it wrote the diff. Give it the skill, and the follow-through stops being your job.

The changelog writes itself

The announcement is drafted from the actual diff — accurate, categorized, published now or scheduled for launch morning. Users find out the day it ships, not whenever someone remembers.

publish a changelog for what we just merged

Feedback answered the day it ships

The people who asked get a reply, items flip to completed, and the public roadmap moves itself — columns are live views over feedback status. No stale board, no ghosted voters.

mark the dark mode feedback completed and reply to voters

Docs by the agent that built it

The best-placed author for a feature's help article is the agent that implemented it. It drafts in the right category, with revisions tracked, ready for your review.

draft a help article for the new export feature

Support triaged before standup

The agent reads the whole queue with your docs and codebase in context — answers what's documented, labels and prioritizes the rest, and leaves reply drafts on the tricky ones.

Getting started

Install once, prompt forever

Add the skill to your agent, export an API key from Settings → API in your Userorbit workspace, and every product chore becomes a one-line prompt.

Read-only keys can call list/info/count/search endpoints; read/write keys can call everything. If your tool speaks MCP, there is also a hosted server at api.userorbit.com/mcp.

1 · Install claude install-skill https://github.com/userorbit/skill
2 · Authenticate export USERORBIT_API_KEY="uo_..."
export USERORBIT_TEAM_ID="..."
3 · Prompt "we shipped dark mode — close the loop"
Prefer MCP? OAuth, no keys to manage claude mcp add --transport http userorbit https://api.userorbit.com/mcp

Why it works

The agent that ships the code closes the loop

A separate "AI PM tool" doesn't know what you just built. Your coding agent does — it wrote the diff. The skill hands it the map of your whole product platform, so it can finish the job it started.
