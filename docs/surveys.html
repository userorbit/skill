Userorbit Skill/Surveys

In-App Surveys

Launch targeted in-app surveys from a prompt.

Surveys let you ask users the right question at the right moment — NPS, CSAT, feature reactions, churn reasons. With the skill, your agent can spin up a survey with questions, targeting, and triggers, publish it, and iterate on the questions later.

This guide covers the agent skill and API. For the full product, see Userorbit surveys.

Ask your agent

  • Create an NPS survey targeting users active for more than 30 days
  • Add a follow-up question to the churn survey asking what almost made them stay
  • Pause the pricing survey

Endpoints

All endpoints are POST https://api.userorbit.com/api/v1/<endpoint> with a Bearer API key and x-team-id header.

EndpointWhat it does
surveys.createCreate with questions, targeting, and design configuration
surveys.list / drafts / all / count / infoBrowse surveys
surveys.updateEdit questions (add/update/remove), targeting, publish state
surveys.deleteRemove a survey

Example

Examplesurveys
curl -s -X POST "https://api.userorbit.com/api/v1/surveys.all" \
  -H "Authorization: Bearer $USERORBIT_API_KEY" \
  -H "Content-Type: application/json" \
  -H "x-team-id: $USERORBIT_TEAM_ID" \
  -d '{"search": "NPS"}'

Full request and response schemas: references/api.md.
